SISTEMA DE COMPRAS Y VENTAS DE UN MICROMERCADO
==============================================


1. INGRESO DE PROVEEDORES
2. LISTADO DE PROVEEDORES
3. BUSQUEDA DE PROVEEDORES POR LETRAS.
4. BORRADO DE PROVEEDORES SI NO SE LES HA COMPRADO.
5. ACTUALIZAR DATOS DE PROVEEDORES.


6. INGRESO DE CLIENTES.
7. BORRADO DE CLIENTES SI NO HAN COMPRADO EN EL MICRO
8. LISTADO DE CLIENTTES.
9. BUSQUEDA DE CLIENTES POR LETRAS.
10. ACTUALIZACION DE DATOS DE CLIENTES.


11. INGRESO DE PRODUCTOS.
12. ACTUALIZACION DE DATOS DE PRODUCTOS.
13. BORRADO DE PRODUCTOS SI NO HAN SIDO COMPRADOS O VENDIDOS.
14. LISTADO DE PRODUCTOS.


COMPRAS
=======

15. REGISTRO DE LAS COMPRAS A NUETROS PROVEEDORES.
    FECHA DE COMPRAS, PROVEEDOR AL QUE SE LE COMPRA.
    NUMERO DE LA COMPRA.

16. EL SISTEMA PERMITE REGISTRAR PARA UNA COMPRA UNO O VARIOS
    ARTICULOS CON SU CANTIDAD. CON EL PRECIO UNITARIO.


17. EL PRECIO DE VENTA AL PUBLICO SE LO REGISTRA A NIVEL DE ARTICULO.


VENTAS
======

18. REGISTRO DE LAS VENTAS A NUESTROS CLIENTES.
    FECHA DE VENTA, NUMERO DE LA FACTURA, DATOS DEL CLIENTE.

19. EL SISTEMA PERMITE REGISTRAR PARA UNA VENTA, UNO O VARIOS
    ARTICULOS, CANTIDAD, PRECIO DE VENTA AL PUBLICO (QUE VIENE DE
    LA LISTA DE PRODUCTOS) .

20. EN UNA VENTA SE CALCULA EL SUBTOTAL, IVA Y TOTAL.




21.EL SISTEMA DEBE MOSTRAR EL  HISTORIAL DE A QUIEN SE LE HA 
   COMPRADO UN PRODUCTO.


22.EL SISTEMA DEBE MOSTRAR EL HISTORIAL DE AQUIEN SE LA HA 
   VENDIDO UN PRODUCTO.

23. EL SISTEMA PERMITE VER UN LISTADO DE TODOS LOS PRODUCTOS Y 
    LAS CANTIDADES QUE SE TIENE EN EXISTENCIA.

24. CUANDO SE COMPRA A UN PROVEEDOR, SE AUMENTA LA EXISTENCIA.

25. CUANDO SE VENDE A UN CLIENTE, DISMINUYE LA EXISTENCIA.
